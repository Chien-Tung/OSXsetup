# OSXsetup

OSXsetup is a script to set up an OS X macbook for web development.

## Requirements

A Macbook.

## How to start

1. Install Xcode and remember open it.
2. Install XCode Command Line Tools `xcode-select --install`
3. `$ cd OSXsetup`
4. `$ bash setup.sh | tee ~/install.log`
